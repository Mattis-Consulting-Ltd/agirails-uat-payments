This project aims to test trusted subcontractor payments based on UAT-approved deliverables using AGIRAILS contracts on the Base blockchain network.

Objective

To automate subcontractor payments in a transparent, auditable way that relies on User Acceptance Testing (UAT) proofs. When deliverables pass acceptance criteria, the escrow contract automatically releases funds to subcontractors.

Scope

• Select a well-bounded project or task within automation delivery workloads.
• Define measurable acceptance criteria and test cases.
• Develop UAT harness scripts to generate contract-proof manifests.
• Deploy escrow contract on Base network.
• Execute subcontracted agents under AGIRAILS framework.
• Automate payment release on UAT pass.
• Record results in Notion knowledgebase.
Success Criteria

• Payments triggered correctly on proof validation.
• Contract audits verified.
• Transparent, traceable logs of subcontracting steps.
• Efficiency gains and risk reduction demonstrated in pilot.
Repo Structure

• /contracts - Solidity smart contracts.
• /middleware - Node.js/TypeScript API for integration bridge.
• /uat-harness - Scripts for UAT harness and manifest generation.
• /ipfs - IPFS integration modules.
• /frontend - Dashboard and portal code.
• /docs - Project documentation and architecture.
